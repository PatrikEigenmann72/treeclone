# treeclone

Who hasn’t used the UNIX tree command to get a quick overview of a directory?
treeclone is a compact re‑implementation of that idea — a small C program that walks your filesystem and prints a clean, readable tree view of folders and files.
It’s fast, minimal, and configurable, with sensible defaults and a simple exclusion system.
If you want a portable, dependency‑free tree tool that behaves exactly the way you expect, treeclone has you covered.
